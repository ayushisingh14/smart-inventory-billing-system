# 📦 Smart Inventory and Billing System

A modern, full-stack inventory management and billing application built with **Python**, **Streamlit**, and **PostgreSQL**. 

This system provides a clean, user-friendly interface to manage customers, track product inventory, process sales, generate bills, and view business analytics in real-time.

## ✨ Features

* **📊 Interactive Dashboard:** Get a quick overview of total customers, active products, and total sales.
* **👥 Customer Management:** Perform full CRUD (Create, Read, Update, Delete) operations on customer profiles.
* **🛒 Product Management:** Track inventory levels, update pricing, and manage product details.
* **🧾 Sales & Billing:** Create new sales, automatically update inventory stock upon purchase, and generate detailed customer bills.
* **📈 Analytics & Reports:** View daily sales trends, filter sales by date range, identify top-selling products, and receive low-stock alerts.

## 🛠️ Tech Stack

* **Frontend / UI:** Streamlit
* **Backend Logic:** Python 3 (Object-Oriented Programming structure)
* **Database:** PostgreSQL (using `psycopg2` for database connections)
